Juego de cartas UNO

Basado en el famoso juego de Matel. ¡El primero en quedarse sin cartas gana!. Juega contra la CPU utilizando cartas de acción como reversa, pierde turno, cambia color, entre otras. Luego de cada partida, el ganador suma puntos, ¡llega a 500 para ganar el juego!.

Caracteristicas

  • Realizado con HTML, CSS y únicamente Javascript
  • Sin ninguna librería
  • Implementa todas las reglas del juego
  • Animaciones
  • Reacciona a las interacciones del usuario
  • Posee sonidos
  • Las cartas son de creación propia
  • Utiliza conceptos de forma creativa para solucionar problemas en la jugabilidad
